# infectious-disease-xai
This project focuses on developing an interpretable and data-driven framework to understand and predict the spread of infectious diseases such as Tuberculosis, along with emerging outbreaks like Monkeypox and evolving forms of COVID-19. Rather than relying solely on traditional epidemiological models, the approach combines stochastic disease modelling with modern artificial intelligence techniques to better capture real-world uncertainty and complex transmission patterns. A key emphasis of the project is on explainability using methods such as feature attribution and interpretable learning architectures. This ensures that the results are not only accurate but also transparent and trustworthy for public health decision-making. By integrating modelling, prediction, and interpretation into a unified framework, the project aims to provide actionable insights for early outbreak detection and predictions.
